# YoYoCinema
App developed using [The Movie Database API](https://developers.themoviedb.org) to search and favorite movies.
# What was used?
  - Kotlin
  - MVVM Pattern
  - Android Architecture Components (LiveData, Room, ViewModel)
  - Retrofit
  - Picasso
  - Gson
  - Offline Support  for favorites
